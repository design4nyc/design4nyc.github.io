Java Developer, Integration Services

Within NYC Opportunity, the Enterprise Data Solutions (EDS) team supports meaningful data integration that allows City agencies to increase efficiency and access to benefits and services. To do this work, we use proven strategies to responsibly bring together data. In addition, the EDS team develops technology and applications that allow for independent rapid on-demand matching capabilities that operate within strict legal frameworks.

The Java Developer will be part of the Integration Services Unit within the EDS team. The Java Developer will maintain and expand digital products, and develop new services that provide effective data solutions for New York City’s complicated data needs.


  Responsibilities

  • Use technology solutions to solve business problems.
  • Contribute to new technology projects as assigned and provide operations & maintenance support for existing applications.
  • Develop custom code and web services that allow seamless integration between systems.
  • Unit test newly developed code and support system testing efforts.
  • Troubleshoot errors, defects and other system issues.
  • Maintain system document and source code repositories.
  • Support the team director and fellow team members as projects are assigned.

  Skills We're Seeking

Proficient knowledge of server-side programming like Java, PHP, Python, Javascript, Ruby.

Proficient knowledge of most common databases such as Oracle, Microsoft SQL Server, MySQL, MongoDB etc.

Experience using and developing APIs and data web services and working with data from diverse sources including case management systems, informational content, and other data feeds.

Experience with ETL programming and data modeling.

Experience developing applications in n-tier environments.

Strong written, verbal and interpersonal skills.

Experience working with cloud computing integration such AWS, Azure or private cloud environments.

Experience with an agile, iterative development process and version control tools such as Git, GitHub, Subversion.

Client focused, committed to rapid execution, and strategic and flexible.

  Education and Experience

Bachelor's degree in technology or equivalent preferred with 3-5 years of professional experience.

If you're a recent graduate of a developer bootcamp, we recommend you check out our Engineering Fellow position.

Successful candidates will be invited for an initial phone screen. Due to the volume of applications received we may not be able to respond to every application.
